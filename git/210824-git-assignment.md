## Key point
- Main에서 spiderman.md를 생성하고 (빈파일) 세 개의 브랜치를 생성한다.
- Format은 영화 하나당 내용이므로, 각 브랜치 내에 영화마다 커밋을 진행한다.
- 작업이 끝난 branch는 main에 머지 후 삭제한다. (혼선 방지)
- Merge 중 conflict 해결을 하면서 내용이 크게 변경되어도 큰 문제는 없다.
